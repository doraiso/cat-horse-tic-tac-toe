# 🐱 vs 🐴 3×3ゲーム

https://doraiso.github.io/cat-horse-tic-tac-toe/

小さな盤面で、  
🐱と🐴が交互にマスを取り合うだけのゲーム。

3×3。  
たった9マス。

先に一列そろえたほうが勝ち。  
そろわなければ、引き分け。

---

## ルール

- 🐱が先手
- 交互に1マスずつ置く
- 縦・横・斜めに3つ並べたら勝ち
- 両者が最善を尽くすと引き分けになる

---

## このゲームについて

三目並べは、理論上すでに解かれているゲームです。  
完璧に打てば、どちらも勝てません。

それでも実装してみると、

- UIの組み立て
- 状態管理
- 勝敗判定ロジック
- CPUの簡易思考

といった、小さな要素がきちんと詰まっています。
